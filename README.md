## Overview
The **Car Rental System** is a comprehensive full-stack web application designed to let users browse and book cars and allow administrators to manage the fleet, bookings, payments, and more. It includes advanced features like an AI chatbot, dynamic pricing, coupons, notifications (email/SMS), image uploads, and invoice generation.
